# WeatherAppArs
Простое и функциональное мобильное приложение для отслеживания погоды в реальном времени. Проект создан в рамках изучения современной Android-разработки с акцентом на чистоту кода и масштабируемость.
🚀 Основные возможности
Текущая погода: получение данных о температуре, влажности и условиях в реальном времени.
Поиск по городам: возможность узнать погоду в любой точке мира.
Адаптивный дизайн: интерфейс корректно отображается на различных экранах.
Работа с API: интеграция с внешним сервисом данных (OpenWeatherMap).
🛠 Стек технологий и архитектура
Проект реализован с использованием современных инструментов:
Язык: Kotlin
Архитектура: MVVM (Model-View-ViewModel) — для четкого разделения логики и интерфейса.
UI: Jetpack Compose — декларативный подход к созданию элементов управления.
Сеть: Retrofit + OkHttp — для эффективной обработки HTTP-запросов.
